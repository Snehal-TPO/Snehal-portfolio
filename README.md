# Snehal Pol Portfolio

Responsive one-page portfolio for Snehal Pol, Technical Product Owner for platform, API, AI systems, and SaaS products.

## Files

- `index.html` - Bootstrap one-page portfolio template.
- `assets/product-owner-hero.png` - Hero visual used by the portfolio.
- `assets/snehal-pol-resume.pdf` - Downloadable resume linked from the page.

## GitHub Pages

Upload the files to a GitHub repository and enable GitHub Pages from the repository settings. The page is static and does not require a build step.
